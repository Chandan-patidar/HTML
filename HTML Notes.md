## 2.3 Basic HTMl Page

<head>                              Parent of meta data
    
    <title>My First Webpage</title>         Title ofthe web pages
</head>
<body>                                      Parent of content tags
    <h1>  Hello world !</h1>                Heading tag
</body>
</html>
